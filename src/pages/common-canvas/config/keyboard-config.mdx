## Keyboard Config object

The keyboard config object configures whether certain actions are available from the keyboard. See this section for what <Link to="/common-canvas/canvas-documentation#keyboard-support">key combinations</Link> are supported

```
const keyboardConfig = {
    actions: {
        delete: false,
        undo: false,
        redo: false,
        selectAll: false,
        cutToClipboard: false,
        copyToClipboard: false,
        pasteFromClipboard: false
};
```

All actions are true by default so it is only necessary to specify those actions you don't want as `false`.
